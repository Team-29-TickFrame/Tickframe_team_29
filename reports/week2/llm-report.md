## AI Tools Used

The team used ChatGPT (OpenAI GPT) during Week 2 activities.

## Usage

LLM tools were used for the following tasks:

* Improving the readability and structure of customer notes.
* Assisting with debugging code and identifying implementation issues.
* Providing general guidance during development and documentation.

## Human Contributions

All project decisions, user stories, prioritization, interface design, and final documentation were reviewed and approved by the team.
The team independently evaluated all AI-generated suggestions before including them in the project.

## Summary

ChatGPT was primarily used as a support tool for meeting transcription, note organization, and code debugging. Final decisions and project outcomes remained under team control.
