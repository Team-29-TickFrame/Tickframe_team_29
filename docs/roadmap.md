# Roadmap

Tickframe's product goal is a real-time crypto market pattern recognition and
metrics engine that provides honest signals, reproducible analysis, and a clean
workflow for team delivery.

## Sprint 1

- Milestone: [Sprint 1](https://github.com/Team-29-TickFrame/Tickframe_team_29/milestone/1)
- Dates: 2026-06-15 to 2026-06-21
- Sprint Goal: deliver the MVP v1 market analytics increment together with the
  repository workflow and documentation required for Assignment 3.
- Focus: Bybit live and historical OHLCV for the ten agreed MVP instruments,
  pattern detection, calibrated signals, quantitative metrics, backtest stats,
  chart visualization, high/low breakout alerts, and the workflow/docs needed
  to submit the release cleanly.
