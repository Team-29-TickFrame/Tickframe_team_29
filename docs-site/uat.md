# User Acceptance Tests

The maintained UAT documentation records reviewer-facing user scenarios for the
Tickframe product. Public UAT notes should describe the scenario, outcome, and
sanitized evidence without publishing private recordings or customer-identifying
details.

## Maintained UAT Areas

- Open the product and inspect the main dashboard.
- Inspect market metrics and analytics output.
- Check latency or system health information.
- Use Sprint 3 UAT scenarios as public notes when they are sanitized and ready.
