v2 / Assignment 5 evidence package. It is generated from maintained repository
documentation and is intended for reviewers who need a browsable view before

## Public Evidence Boundary

This site contains only public, sanitized documentation. Private recordings,
exact private timecodes, credentials, university email addresses, and
customer-identifying details belong only in private submission channels.
